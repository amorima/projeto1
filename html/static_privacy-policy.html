Política de Privacidade

A Hoot compromete-se a proteger a sua privacidade. Esta política descreve como recolhemos, utilizamos e protegemos os seus dados pessoais.

Dados Recolhidos

  • Informações de contacto (nome, email, telefone)
  • Dados de reserva e preferências de viagem
  • Informações de pagamento (processadas de forma segura)
  • Dados de utilização do website

Utilização dos Dados

Os seus dados são utilizados para processar reservas, melhorar os nossos serviços, comunicar consigo e cumprir obrigações legais.

Partilha de Dados

Podemos partilhar os seus dados com parceiros de viagem necessários para completar a sua reserva, sempre em conformidade com a legislação aplicável.

Os Seus Direitos

  • Acesso aos seus dados pessoais
  • Retificação de dados incorretos
  • Eliminação dos seus dados
  • Portabilidade dos dados
  • Oposição ao tratamento
